### tinyMegaCore Installation
- Add board manager url to preferences
    - http://drazzy.com/package_drazzy.com_index.json
- Search for "megaTinyCore" and install

### Options to Change
- Change board to mcu being used
- Change chip to mcu being used
- Change clock to "10MHz internal" if using battery power
- Change programmer to "SerialUPDI - SLOW: 57600 baud"
